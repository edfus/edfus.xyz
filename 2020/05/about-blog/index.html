About-博客

这里就是我的博客啦。


……有很多的想说的，可想想还是不说比较好。想表达的越多、细节越完善、越重视，我这个人越是思虑、斟酌乃至拖延。最近意识到的老毛病。


总之，本博客由github pages托管，使用Hexo生成，主题由Hexo内置主题landscape修改而来。

域名方面，本站使用顶级域名xyz。注意edfus.eu、edfus.com并非我注册、与我无关。

说起来自己用了好些年的网名竟被Europen Database For Ukrainian Seafarers给后来居上再解构了……莫名有些好笑。不过我觉得edfus.eu、edfus.com、edfus.xyz中 edfus.xyz 是最漂亮的一个（笑）（也最便宜XD），而Europen Database For Ukrainian Seafarers不也挺不错？举双手支持！

出于个人审美考虑，本站不添加三级域名www。但www.edfus.xyz和edfus.github.io仍可访问（并自动跳转至edfus.xyz）


在网速优化方面，因为是托管在github pages上的网站，国内访问速度可能较捉急，但jQuery库默认的是百度的源，因此没准是国内国外差不多慢——大家都一样慢，听着就很油库里。

网站几乎都是静态页面，HTML、CSS、JS文件都有通过gulp压缩，JS和图片使用jsdelivr的免费Github CDN服务，因此总的访问体验应该还是不错的。

浏览器兼容。虽然有注意一些明显的兼容性问题，但我这儿全程使用Chrome开发，也没在其他浏览器上测试过，可不知道实际效果咋样。不过样式方面我是用Stylus来编写生成的，CSS兼容性应该挺不错。预处理器牛逼！

关于网站设计……

网站的icon为Takumi老师画的绝对许早苗(さなえアフター)。刚开始着手建站时正好看到了，效果挺好。

网站以暗色调为主，且不说是否利于阅读，至少足够出众（？）。

嘛，我一开始在网站视觉展示上追求的即我书写时的展示与在网页上的展示相同（所见即所得，这也是我选择Typora的原因）。因此这个网站的设计基础是我的Typora主题，而我的Typora主题是由Typora默认主题（之一）night魔改而来的——这个网站的主色调其实相当随意地决定于我当初上手Typora时的心情……

不过我觉得展示效果不差，我平日里用Typora写md的时候也挺舒服的。

博客不少元素、颜色都是从许许多多的前辈网站那里拿来的，比如文章的强调颜色SAMPLE来自SALLY，网页“gotop”设计借鉴于幻奏盛宴……来源我大致都有“标记”。感谢前辈们。

网站结构方面，我这网站虽然是通过Hexo生成的，但在各种折腾后大概与传统Hexo博客大相径庭了。比如这儿的结构设计是以分类为导向的，而大部分Hexo博客则更倾向于时间。我喜欢分类的爽利惬意。

主页方面主要参考SAi的博客的时间流展示形式，而分类页的category + tags的形式的灵感来自Hux Blog，more页面的设计参考Typora主题文档页面。核心考虑是媒介对信息展示的影响。

网站是响应式设计，虽然我不推荐手机端浏览，但手机端相关的优化我都有考虑，效果应该不错（虽然冇自己体验过）。手机端的Banner是米山舞老师为RADIO EVA绘制的宣传插画，当初测试CDN效果的时候看着这张图比较大就用了，效果还不错。

up: 后来换了Banner，极端化了手机端展示。

另外，主页、文章页和特定屏幕尺寸下的分类页的“Search”按钮仅仅是谷歌搜索的便捷式跳转，我未做SEO，未向任何搜索引擎提交Site Map，未建立Robots.txt，也没有使用谷歌统计之类的服务（顺便一提），因此不保证满意的搜索结果，但也许能搜到一些我删掉的东西。

up: 后来做了SEO。

在交互方面，我没有保留RSS——RSS干着将信息从主人家的支配、氛围中肢解刨除一般的事，这于我对这个网站的期望是相反的。虽然我挺喜欢用RSS就是。

评论使用gitTalk插件，即借用github的issues来盛装评论。我大概只会在tech分类的文章中开放评论。

如果想联系我的话，请发邮件到edfus@edfus.xyz。

关于我期望这个博客上有什么。

我是为了什么而建这个博客的呢？想在这个博客上展示什么呢？希望什么人看到呢？……

我是互联网一代。我的人生观、价值观、世界观来自于互联网与书籍，来自他人的故事、遐想、思考、总结。 我最初产生想建博客的想法即是因为SAi的博客。

……把自己的故事、遐想、思考、总结分享于互联网上，让某个和自己类似的人的看到，让ta能有些收获、或是单单知道“You are not alone”。或者提示、警示些什么…… 成为互联网分享圈子的一环，帮助他人，良性循环……

但我清清楚楚地知道以上都是假的，无论以前还是现在。

我很喜欢EVA第二十六话标题“在世界中心呼唤爱的怪兽”（不论同名科幻小说）。怪兽们是畸形的、扭曲的。他们渴望得到爱，却又自顾自地否定自己能得到爱的可能，用怪兽般的外在将自己武装，内心深处怯懦自卑、与根本上恐惧他人、极度敏感于伤害……“只要他人无法察觉到真实的“我”之模样，那“我”自然永远无法被真实地伤害”；“没人能打倒一个已经被打倒了的人”；“只要能将自己置于道德的、现实利害的高处，那自然不是“我”错了，而是他人错了”…… 对于怪兽们来说，任何的他人都是异样的、与自己不同不相容的存在，他们的世界是狭隘的，里面只有他们自己、他们那在逼仄中不断反射的嘶吼，与飘忽的他人之幻象。他们站在世界中心，恐惧于挪出仅仅一步，他们渴望爱却又披上了怪兽的武装，身为怪兽却大声呼喊。到头来，他们所做的不过只是在伤害自己和接近他们的人——虽然他们成为畸形怪兽并不是他们本身的错。怪兽只该在暗芜处伏匿，或者设法成为人。

呼……说的有些偏了。

总之，我若在这博客上摆上我的“思考”，美名其曰分享，实际上不过是希望告诉大家或某人我多么有深度，平日里思考了多么多，自己的行动力有多么强、克服多少东西，真是牛逼——希望得到认同、喜爱、赞赏。或是在这狭小空间内埋下我的深层，当现实中不被理解、认知失调时暗自嗤笑“你懂我什么”，并幻想别人在看到这么个牛逼的空间后“竟恐怖如斯”。

换句话说，因为我的错误人格、错误展示方式、错误期望，“在某个地方上展示自己的好”这种普世价值观并不排斥的做法变成了一种高级自慰，手冲TWO POINT OH。

另外，对于我这样的小意、现实、自私、强调理性的人，对于如今有丰富多彩的自我价值实现方式的自闭青年我来说，我真的会以这个没流量的可怜网站所能传达的人群中的某个你的可能的或许只持续一小会儿的不知道到底是因为什么的感慨视作能战胜我之附骨羞耻感的乐吗？……这样的惺惺作态比酷热天把猪油涂满全身后徒步慢走20公里那蠕动粘黏抽丝颤震蜿蜒堆积溢漏还要恶心、令人不寒而栗。

……我决定不把上面这些上头话删掉。我最喜欢讽刺小鬼了，特别是那种自己一不小心就会成为的小鬼。


不过有着这么个博客还是挺棒的，因为人生总得找些事来做嘛（笑）

我非常喜欢并有意识地以表达的形式来整理思绪、理顺逻辑（审阅）、“突发”灵感（发现）、弥补一人思维的不足、验证是否我的想法从一开始的源头就错了（被挑战并内化之），我也不缺表达的对象。但随着表达的次数越来越多，我却发现尽管自己的表达能力越来越精进，自己在表达时所受到的挑战、思维碰撞竟越来越少，自己在表达过程中越来越像是在唱独角戏。

……我想，随着我个人的不断进步，表达的内容自然越来越深、细节也越来越多，阐明更加难、更花时间（在另一方保持静态的条件下）；而我的朋友们也从思维深处逐渐习惯于从我这接受，辩驳的欲望枯黄了；或者，随着我对我的生活方式的不断审视、反思、优化，我的思考时间与质量与范围渐进，而表达时间与之相比显得太少……以及很多很多的小原因们。

我的朋友们都很棒，遗憾的是他们并没有和我走在同一条道上，我们间的沟壑已始显形。但这本就是早晚的事——没有人能承载你的全部，除了你自己。另外，我无法、也懒于预测将来自己会遇到哪些人、有怎样的机会。真正的强者不将希望托付于他人身上，却也不否认托付的可能、不丧失自己托付的能力。

因此，我希冀于写作这样的方式能更切实地有效率地帮助我思考、增加我的成就感与动力，强调那些值得强调的东西，不丧失掉那些弥足珍贵的东西。

不过，书写也是一个比思考更耗费精力、时间——特别是时间——的东西。我现在还不具有能将一些东西理性条理地诉诸纸面的余裕。

总之，我希望这个博客里能记录一些我和我的朋友们没有谈到过的东西，一些写起来简单容易、不大会让我上头的东西，一些的视觉性的或总结性质的东西，一些我不会特别想表达、想让某些人知道的东西（该思考该写，但不该声扬）——不管怎样，书写博客这样对着旷野群山大喊似的行为都不应该替代现实的表达和对现实的力比多投入。嘛，要是响子酱跳出来、快乐地大喊回应的话倒是可以立马替代掉（笑）


另外我的一个毛病是想的比起应表达的多太多了，写文章写到后面就上头、开始掺入各种各样的“噪点”了。也希望这点能在写博客的过程中克服掉。


博客的企划始于19年11月，20年1月末开始细化改造，大体完成于20年2月下旬。本文写于同年5月27日。

表达、行动，认知失调，思考内化。
